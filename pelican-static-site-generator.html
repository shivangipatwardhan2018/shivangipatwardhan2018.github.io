Pelican Static Site Generator

Mon 25 December 2017

Introduction

Pelican is a python-based static site generator that is a wonderful tool for technologies that are not very committed to front-end developement. Pelican allows users to deploy sites quickly (30 minutes) and effectively. There is a wonderful post about deploying your first pelican website found here.

The process of deploying the website is quite simple. I created a rundeploy script to expidiate the deployment process and handles the gerenation and local deployment of the website.

pelican content &&
cd ./output
python -m pelican.server
